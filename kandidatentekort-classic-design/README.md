# Kandidatentekort.nl - Meta Ads Landing Page

Dark gradient landing page voor Meta Ads campagnes met:
- ✅ Dark gradient design (#0f172a to #1e293b)
- ✅ Correct Facebook Pixel ID: 238226887541404
- ✅ Typeform integratie (kalFRTCA)
- ✅ UTM parameter tracking
- ✅ Mobile responsive design

## Deploy instructies

### Optie 1: Direct Netlify Drop
1. Ga naar https://app.netlify.com/drop
2. Sleep deze hele folder naar de browser
3. Koppel aan kandidatentekortv2 domein

### Optie 2: GitHub Deploy
1. Push naar GitHub repository
2. Koppel in Netlify aan kandidatentekortv2
3. Deploy automatisch
