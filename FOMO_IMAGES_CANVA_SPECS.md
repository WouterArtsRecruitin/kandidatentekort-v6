# FOMO Images for Kandidatentekort Campaign - Canva Design Specifications

## General Specifications for All Images
- **Dimensions**: 1200x628px (Facebook Ad optimal size)
- **File Format**: PNG or JPG (high quality)
- **Mobile-First Design**: Text must be readable on small screens

## IMAGE 1: COLD AUDIENCE - "Het Geld Dat Je Laat Liggen"

### Design Elements:
- **Background**: Solid black (#000000)
- **Main Text**: 
  - "€500 PER DAG" 
  - Font: Bebas Neue or Impact (bold, condensed)
  - Size: 180-200pt
  - Color: Bright red (#FF0000)
  - Position: Center-top
  
- **Secondary Text**:
  - "47 DAGEN = €23.500 WEG"
  - Font: Montserrat Bold or Arial Black
  - Size: 60-70pt
  - Color: White (#FFFFFF)
  - Position: Center-bottom
  
- **Visual Elements**:
  - Add money icons falling/fading away (opacity 20-30%)
  - Small calendar icon next to "47 DAGEN"
  
### Canva Instructions:
1. Create new design → Facebook Ad (1200x628px)
2. Set background to black
3. Add text elements with specified styling
4. Search "money falling" in elements for icons
5. Apply drop shadow to main text for depth

## Pro Tips for Canva Creation:

1. **Use Brand Kit**: Set up Recruitin brand colors
   - Primary: #FF6B35 (Orange)
   - Urgent: #FF0000 (Red)
   - Success: #10B981 (Green)
   - Dark: #000000 (Black)

2. **Effects to Apply**:
   - Drop shadows on all text
   - Glow effects on urgent elements
   - Slight blur on background elements

3. **Testing**:
   - Preview on mobile view
   - Ensure 20% text rule compliance
   - Export at highest quality

4. **Alternative Fonts in Canva**:
   - Anton (instead of Bebas Neue)
   - Oswald (for headlines)
   - Open Sans (for smaller text)

## Quick Creation Workflow:

1. Open Canva → Create Design → Facebook Ad
2. Set background first
3. Add largest text elements
4. Layer secondary elements
5. Apply effects and shadows
6. Add urgency indicators (timers, arrows)
7. Export as PNG/JPG

## Facebook Ads Manager Upload:

1. Export all three images at high quality

   - kandidatentekort_warm_127bedrijven.png
